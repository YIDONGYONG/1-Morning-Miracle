🌅 1 Morning Miracle

意志力ではなく「仕組み（システム）」によって朝の行動を変革

📱 プロジェクト概要

朝の習慣を自動化し、目標達成を支援する朝専用の習慣化アプリです。 意志力に頼らず、システムとして朝のルーティンを実装します。

✨ 主な機能

🎯 朝のルーティン管理

毎朝のタスクを自動管理し、習慣化をサポート

📊 進捗トラッキング

継続状況をビジュアル化し、モチベーション維持

🔔 スマート通知

最適な時間にタスク通知をお届け

⚙️ システム設計

意志力に頼らない仕組みで継続を実現

🚀 はじめに

詳細は GitHub リポジトリ をご覧ください。

📝 プロジェクト進行

このプロジェクトは 1課題 = 1 Pull Request の形式で進めます。
各課題は以下の流れで進行します：

  • ✅ 新しいブランチを作成
  • ✅ 課題を実装
  • ✅ Pull Request を作成
  • ✅ レビューを受ける
  • ✅ main ブランチにマージ
